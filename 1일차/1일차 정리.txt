const : 바뀌지 않은 값
- const로 변수를 설정하면 값을 바꿀 수 없다
ex)
const a = 5;
 console.log(a + 2);
 console.log(a * 2);
 console.log(a / 2);
but, const가 없다면
 console.log(5 + 2);
 console.log(5 * 2);
 console.log(5 / 2);
숫자를 변경할 때 6번을 바꿔야한다.
const 설정을 통해 한번에 const에 담긴 값만 변경하면 따로 변경하지않아도 된다.

const myName = "nico";
- const 값을 설정할 때 다른 단어를 사용할 때는 대문자를 사용하여 구분할 수 있도록 하는 것이 편리하다.
모두 소문자로 작성해도 에러를 일으키지는 않지만, 같이 일하는 사람들이 불편함을 느낄 수 있다.
이를 camelcase라고 한다.
